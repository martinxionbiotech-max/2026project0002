# Car Wrap Calculator

A static web app to estimate vehicle wrap area, required roll length, roll count, and material cost.

## Run locally

```bash
python -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

## GitHub Pages one-click publish

The workflow file is located at `.github/workflows/pages.yml`.

1. Go to **Settings → Pages** and choose **GitHub Actions** as the source.
2. Push to `work` or `main`, or run the workflow manually from the Actions tab.
3. Access your site at: `https://<your-github-username>.github.io/<repo-name>/`.
